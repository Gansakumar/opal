Welcome

What authors say:

“Enjoy trying a tool that optimizes an algorithm in a snap!”

OPAL stands for Optimization of Algorithm. OPAL provides a framework in which algorithmic parameter optimization problems can be defined and solved in a programmatic way.

Our approach is formulate a parameter optimizaiton problem as a blackbox optimization problem. This is so different from the popular approaches that uses heuristic to minimize the running time as unique goal. The formulated blackbox has the objective function and constraints built from parameter values and elementary measures obtained from empirical tests. Hence, the users can optimize his algorithm in any aspect and exploit any algorithmic parameter. The blackbox problem is, in general, solved by any direct-search solver

OPAL is written in Python and uses Python as language to describe the parameter optimization procedure that consists of three main steps: describle the target algorithm, define optimization problem and choose a suitable solver.
